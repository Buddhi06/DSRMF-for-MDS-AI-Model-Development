Task 04: - Identify the sources of information to be used as inputs to the assessment

    Next

Organisation should identify the sources of descriptive threat, vulnerability, and impact information to be used in the risk assessment [2].

"The framework proposes to use the comprehensive list of assets, threats, vulnerabilities, controls and control implementation details provided in the framework itself as the sources of information."

“The framework also provides a detailed list of references that are used for the identification of threats, vulnerabilities, controls and control implementation details. Moreover, it suggests referring any relevant and current standards, technical reports, scientific papers, websites, online databases and blog posts during the framework implementation to identify any threats, vulnerabilities, risk controls and control implementation guidelines that are not included in the framework.”
